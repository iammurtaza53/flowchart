Routing Layout

There are 2 type of layouts:
  • Full Layout
  • Blank Layout

Full Layout

Full Layout has default 5 components which are Header, Sidebar, Footer, Customizer. When you create any components you have to link in 2 Files, one is sidebarlink.js and another is router.js. it will take as full layout when you import fulllayout component in router.js

Router.js
                                
                                <script>
                                // This is for the scroll top when click on any router link
                                routes: [
                                {
                                path: '/',
                                redirect: 'starterpage',
                                component: () => import('@/layouts/Layout'),
                                children: [
                                {
                                name: 'StarterPage',
                                path: 'starterpage',
                                component: () => import('@/views/StarterPage'),
                                },
                                ]
                                },
                                
                                {
                                path: '/authentication',
                                component: () => import('@/layouts/Blanklayout'),
                                children: [
                                {
                                name: 'BoxedLogin',
                                path: 'boxedlogin',
                                component: () => import('@/views/authentication/BoxedLogin'),
                                },
                                {
                                name: 'FullLogin',
                                path: 'fulllogin',
                                component: () => import('@/views/authentication/FullLogin'),
                                },
                                {
                                name: 'BoxedRegister',
                                path: 'boxedregister',
                                component: () => import('@/views/authentication/BoxedRegister'),
                                },
                                {
                                name: 'FullRegister',
                                path: 'fullregister',
                                component: () => import('@/views/authentication/FullRegister'),
                                },
                                {
                                name: 'Error',
                                path: 'error',
                                component: () => import('@/views/authentication/Error'),
                                },
                                ]
                                },
                                ],
                                </script>

Blank Layout

Blank layout has no default components. It is normally used for Login-Logout page.

Router.js

                                routes: [
                                {
                                path: '/authentication',
                                component: () => import('@/layouts/Blanklayout'),
                                children: [
