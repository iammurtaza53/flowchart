Theme Breadcrumb Adding Breadcrumb into your page

Go to your file in whichyou want to add breadcrumb and For adding breadcrumb add some code after container of that page and add data into the script as shown below:

Dashboard.vue breadcrumb: 

                                
                                <BaseBreadcrumb :title="page.title" :icon="page.icon"
                                :breadcrumbs="breadcrumbs"></BaseBreadcrumb>

In Script tag breadcrumb: 

                                
                                  <script>
                                    export default {
                                    name: "Analytical",
                                    
                                    data: () => ({
                                    page: {
                                    title: "Analytical"
                                    },
                                    breadcrumbs: [
                                    {
                                    text: "Dashboard",
                                    disabled: false,
                                    to: '#'
                                    },
                                    {
                                    text: "Analytical",
                                    disabled: true
                                    }
                                    ]
                                    }),
                                    ]
                                  </script>
