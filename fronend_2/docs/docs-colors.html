Colors Color Defaults

Out of the box you get access to all colors in the Material Design spec through sass and javascript. These values can be used within your style sheets, your component files and on actual components via the color class system.

For each color, we think has its functionality in the application as alerts of errors (danger), warnings to the user (warning), communications of approval (success), or main color of the application (info).

Example

  • info
  • Success
  • Danger
  • Warning
  • Dark
                                
                                        <!-- Change colors in button -->
                                        <v-btn small color="primary">Primary</v-btn>

                                        <!-- Change colors in text -->
                                        <h3 class="info--text">Text</h3>

                                        <!-- Change colors in background -->
                                        <div class="info">Background</div>
