Form Elements

1. Autocompletes

The v-autocomplete component offers simple and flexible type-ahead functionality. This is useful when searching large sets of data or even dynamically requesting information from an API.
Link Here:https://vuetifyjs.com/en/components/autocompletes/

2. Combobox

The v-combobox component is a v-autocomplete that allows the user to enter values that do not exist within the provided items. Created items will be returned as strings.
Link Here:https://vuetifyjs.com/en/components/combobox/

3. File inputs

The v-file-input component is a specialized input that provides a clean interface for selecting files, showing detailed selection information and upload progress. It is meant to be a direct replacement for a standard file input.
Link Here:https://vuetifyjs.com/en/components/file-inputs/

4. Forms

When it comes to form validation, Vuetify has a multitude of integrations and baked in functionality. Want to use a 3rd party validation plugin? Out of the box you can use Vee-validate and vuelidate.
Link Here:https://vuetifyjs.com/en/components/forms/

5. Inputs

The v-input component gives you a baseline to create your own custom inputs. It consists of a prepend/append slot, messages, and a default slot. It is recommended that you extend this component, but it can be used as a standalone.
Link Here:https://vuetifyjs.com/en/components/inputs/

7. Overflow buttons

v-overflow-btn is used to give the user the ability to select items from the list. It has 3 variations: editable, overflow and segmented
Link Here:https://vuetifyjs.com/en/components/overflow-btns/

8. Selects

Select fields components are used for collecting user provided information from a list of options.
Link Here:https://vuetifyjs.com/en/components/selects/

9. Selection controls

Selection control components allow a user to select options. These components must be used with the v-model prop as they do not maintain their own state.
Link Here:https://vuetifyjs.com/en/components/selection-controls/

10. Sliders

The v-slider component is a better visualization of the number input. It is used for gathering numerical user data.
Link Here:https://vuetifyjs.com/en/components/sliders/

11. Textareas

Textarea components are used for collecting large amounts of textual data.
Link Here:https://vuetifyjs.com/en/components/textarea/

12. Text fields

Text fields components are used for collecting user provided information.
Link Here:https://vuetifyjs.com/en/components/text-fields/
